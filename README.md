# This is Group FiveGuys' 3902 Legend of Zelda Project

**Legend of Zelda**\
W to move forward\
A to move left\
S to move back\
D to move right\
N to use attack 1\
M to use attack 2\
E to take damage and die\
O and P to cycle enemies\
U and I to cycle Items\
T and Y to cycle Blocks\
R to Reset\
0 to Quit\
Enter to Pause / Menu
